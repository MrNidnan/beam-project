# User Manual: Start Here

Beam-project helps you show live song information on a projector, TV, second monitor, or browser display.

This manual is written for DJs, teachers and organizers, not developers.

![Screenshot: Beam running with the main window and the display window visible](../docs/images/user-manual/beam_0_hero.jpeg)

## What You Need

Before using Beam, you usually need:

- Beam installed and running
- a supported music player
- one screen for the DJ
- optionally a second screen, TV, projector, or browser device for the audience display

## What Success Looks Like

A basic Beam setup is working when:

- your music player is selected in Beam
- Beam shows the current song in the preview
- the display window shows the same information on the output screen
