# User Manual: First Run

This page helps you get Beam working for the first time.

## Step 1: Start Beam

Open Beam.

You should see the main window with a preview area and settings controls.

If the window opens, Beam itself is running correctly.

## Step 2: Choose Your Music Player

Open the Beam settings and select the music player you actually use.

Common choices are:

- Foobar2000
- VirtualDJ
- JRiver
- Mixxx
- MediaMonkey

## Step 3: Apply the Settings

Click `Apply`.

This saves your settings and tells Beam which player it should read from.

## Step 4: Start Music in Your Player

Play a track in your music player.

Return to Beam and look at the preview.

If Beam is connected correctly, you should now see the current song information.

## Step 5: Open the Display Window

Use the `Display` button to open the audience display window.

Move that window to your projector, second screen, or TV.

If you want full screen mode, use `F11`.

## Step 6: Confirm Everything Before a Live Event

Before using Beam at a milonga or event, check:

- the correct player is selected
- the current song appears in the preview
- the display is on the correct screen
- the text is readable from a distance

## If It Does Not Work Yet

Go to [User Manual - Troubleshooting.md](User%20Manual%20-%20Troubleshooting.md).
